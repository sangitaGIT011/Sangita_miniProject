# Gold Price Analysis and Prediction

##  Overview
This project analyzes historical gold price data and predicts future gold prices using Machine Learning.  
The prediction model is built using the **Random Forest Regressor** algorithm.

---

##  Features
- Gold price trend analysis  
- Correlation heatmap  
- Gold price prediction  
- Accuracy evaluation  

---

##  Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

##  Dataset
Historical gold price dataset from **2016 to 2026**.  
**Dataset Source:** [Gold Price CSV](https://raw.githubusercontent.com/sangitaGIT011/Myproj/refs/heads/main/Gold_Price_2016to2026.csv)

---

##  Machine Learning Model
- **Algorithm:** Random Forest Regressor  
- Purpose: Predicting future gold prices with high accuracy  

---

##  Output
- Gold Price Trend Graph  
- Heatmap Visualization  
- Predicted Gold Prices  
- R² Accuracy Score  

**Example Results:**  
- R² Score: `0.9997`  
- Future Predicted Gold Price: `439.48`
